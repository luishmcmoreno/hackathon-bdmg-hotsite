Registro

Este foi o passo que você já realizou quando inseriu as informações do seu negócio, visualizou a estimativa do investimento e aprovou o inicio da cotação do empréstimo.

Integrador

O integrador é a pessoa responsável por analisar a viabilidade do projeto de instalação fotovoltaica na sua empresa e montar o projeto que será enviado para analise pelo BDMG. Você precisa selecionar um dos intermediadores parceiros para que este passo seja concluído.

Projeto

O integrador é o responsável por montar o seu projeto de instalação fotovoltaica. Para que esta etapa seja concluída o integrador deverá realizar a visita e anexar os documentos descritivos do projeto para análise do BDMG. Caso o integrador demore para a construção e envio deste documento você pode escolher outro integrador ao cancelar este projeto.

Análise

Esta etapa só será iniciada quando todos os documentos do projeto tiverem sido anexados pelo integrador, neste ponto o BDMG irá analisar a viabilidade do projeto e conceder ou não a autorização do desenvolvimento do projeto.

Aprovação

Parabéns, se você chegou neste ponto o seu projeto foi aprovado pelo BDMG e o seu integrador já entrará em contato para o desenvolvimento do projeto.

Não entendi, me explique melhor

Encontre o melhor integrador de acordo com o seu gosto e marque uma visita com ele.
